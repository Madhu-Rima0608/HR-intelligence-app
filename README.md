# 💼 HR Intelligence & Attrition Prediction System

An end-to-end **HR Analytics Web Application** built using Streamlit that helps organizations analyze employee data, identify attrition drivers, and make data-driven HR decisions.

---

🚀 Live Demo

🔗 https://hr-intelligence-app-8vaps87nkhjvvy79fcv7wl.streamlit.app/

---

## 📌 Key Features

### 🔐 User Authentication

* Login system with username & password
* Default password with option to change
* Logout functionality

### 📊 Attrition Analysis Dashboard

* Overall attrition rate, employee count, salary & performance metrics
* Attrition breakdown by:

  * Age group
  * Salary group
  * Tenure
  * Department
  * Job Role
  * Gender
* Business-friendly insights & risk alerts

### 🎯 Departmental Performance

* Performance distribution across departments
* Job role-wise performance comparison

### 🏢 Departmental Productivity

* Salary-based productivity insights
* Department-level comparisons

### 📈 Trend Analysis

* Monthly attrition trends (if date column available)

### 🧠 Advanced Insights

* Attrition vs performance analysis
* Salary-based risk segmentation

### ⚠️ Attrition Predictor (ML Model)

* Built using Random Forest Classifier
* Predicts employee attrition risk based on:

  * Age
  * Salary
  * Experience
  * Performance
  * Job Satisfaction
  * Work-Life Balance
  * Overtime

### 📥 Export Reports

* Download Excel reports with key attrition drivers

### 💬 Feedback System

* Users can submit feedback for continuous improvement

## ⚙️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/hr-intelligence-app.git
```

2. Navigate to the folder:

```
cd hr-intelligence-app
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```
